 🌌 3D Solar System Simulation

A real-time 3D Solar System simulation developed using **Python**, **PyOpenGL**, and **GLUT** for a Computer Graphics project.
 Features

- ☀️ Sun at the center of the solar system
- 🪐 All eight planets (Mercury to Neptune)
- 🔄 Planet revolution around the Sun
- 🌍 Individual planet rotation on their own axes
- 💍 Saturn with rings
- ⭐ Randomly generated star field
- ✨ Orion and Big Dipper constellations
- 🎥 Interactive zoom controls
- 🏷️ Planet labels
- ⚡ Smooth real-time animation

Technologies Used:

- Python 3
- PyOpenGL
- OpenGL
- GLUT / FreeGLUT
 Installation

Install the required libraries:

```bash
pip install PyOpenGL PyOpenGL_accelerate
```

Run the project:

```bash
python solarsystem.py
```

 Controls

| Key | Action |
|------|--------|
| `+` | Zoom In |
| `-` | Zoom Out |

 Computer Graphics Concepts

- 3D Transformations
- Perspective Projection
- Hierarchical Modeling
- Rotation and Translation
- Animation
- Depth Testing
- Double Buffering
